# Number-Checker-Machine-
This is a command-line based number analysis program that checks various mathematical properties of an input number such as:
- Magic Number
- Prime Number
- Sunny Number
- Palindrome Number
- Strange Number
- Adam Number
- Strong Number
- Armstrong Number
- Duck Number
- Automorphic Number
